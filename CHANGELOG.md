# Changelog

All notable changes to this project will be documented in this file.

## [0.0.3] — 2026

### Added
- VS Code extension with custom editor for markdown plan review
- Copilot CLI plugin with `review` skill
- CLI tool (`npx markdown-review`) for browser-based review
- Web app with drag-and-drop markdown upload
- Embedded AI question blocks (open, choice, checkbox)
- Inline annotations on any content block
- General feedback with comment sheet
- Feedback export to markdown and JSON
- Syntax highlighting via Shiki
- Mermaid diagram rendering
- Responsive layout (mobile, tablet, desktop)
- Dark theme

### Security
- XSS protection via rehype-sanitize
- Content Security Policy on all surfaces
